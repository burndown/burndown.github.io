Ultra Mobile Pay as You Go 在中国激活 WIFI Calling

2022-11-07

本文简单介绍了在国内激活 Ultra Mobile Pay as You Go 电话卡 WIFI Calling 的网络设置。

背景介绍

淘宝入手一张 Ultra Mobile Pay as You Go 手机卡，主要用来注册国外服务，接收短信。月租3美元，包含100分钟通话和100条短信。可以在大陆漫游，漫游期间无法通过运营商网络使用套餐的电话和短信，但是可以通过 WIFI Calling 使用套餐内的电话和短信。

WIFI Calling 是一项电信增值服务，手机可以通过家用或者公共 WIFI 接打电话或收发短信，是对运营商网络覆盖薄弱地区的补充。WIFI Calling 需要手机和运营商同时支持才能使用。Ultra Mobile支持这项服务，所以只要你手机支持 WIFI Calling，在运营商侧和手机侧激活了 WIFI Calling 后，就可以使用这张电话卡，在大陆地区通过 WIFI Calling 网络接收短信。

使用这张电话卡，激活 WIFI Calling 是关键，没有激活 WIFI Calling 的手机卡是无法接收短信的。激活 WIFI Calling 需要手机通过 WIFI 去访问 T-Mobile 的 WIFI Calling 服务器，但是这个服务器的因为各种原因，在国内经常无法访问。淘宝上卖这种电话卡的商家基本都提供了代激活功能，但是你也可以自己动手操作。

激活 WIFI Calling 的方法

以下办法假定你已经激活运营商侧和手机侧的网络，只是无法连接到 WIFI Calling 服务器。请参考 IOS 手册去激活手机侧的 WIFI Calling， 请登陆运营商网站或者联系淘宝卖家获取激活运营商侧的 WIFI Calling 指南。

激活 WIFI Calling 的关键是手机能通过 WIFI 访问到 WIFI Calling 服务器。这里包含两步，第一，DNS 能够正常解析到 WIFI Calling 服务器的 (下文简称服务器) IP 地址，第二，手机能通过 WIFI 访问到服务器的 IP 地址，建立连接。通过测试发现，服务器 IP 地址在国内能够正常访问，但是 DNS 不能正常解析服务器的域名。所以解决办法就是使用一个可以正常解析服务器域名的 DNS。

激活 WIFI Calling 的两种办法：

  • 手机上手动设置 DNS：手动设置一个好用的 DNS
  • 路由器上设置域名劫持：在路由器上直接添加服务器的域名和 IP 地址

如果路由器上有代理

如果路由器上有科学上网代理，会稍微麻烦一点，因为服务器地址是国外地址，会走代理，导致无法正常连接和激活WIFI Calling，把服务器地址加入不代理名单即可。

总结

两种方式可以用来激活 WIFI Calling 服务和连接 WIFI Calling 网络：手机上手动设置 DNS 和路由器上添加 WIFI Calling 服务器域名与 IP 对应关系。
